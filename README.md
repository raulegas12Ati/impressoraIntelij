O projeto Imprime um texto, ou um QRCode ou um codigo de barras na impressora Elgin de acordo com os comandos do usuario.
O projeto funciona com base nas funções:
configurarConexao()//Define os dados para que a conexão exista.
abrirConexao()//Abre a conexão 
fecharConexao()//Fecha a conexão 
ImpressaoTexto()// Define os parametros para imprimir o texto 
Corte()//Corta o papel 
ImpressaoQRCode()//Define os dados necessarios para a impressão do QRCOde 
ImpressaoCodigoBarras()//Define os dados necessarios para a impressão do codigo de barras 
SinalSonoro()//Ativa o sinal sonoro da impressora 
ImprimeXMLSAT()// 
ImprimeXMLCancelamentoSAT()// 
respostaDasFuncoesImprimeXML()// 
avancaPapel()//Avança o papel para o corte 
abrirGavetaElgin()//Avança a gaveta 
abrirGaveta()//Abre a gaveta

Funcionamento do sistema: O sistema chama as funções acima de acordo com o que o usuario escolhe no menu.
